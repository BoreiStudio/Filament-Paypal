# Filament PayPal Plugin

A comprehensive Filament v5 plugin for PayPal integration — Orders, Payments, Refunds, Subscriptions, Webhooks, Payouts, and more.

## Features

| Module | Description |
|---|---|
| **Orders** | Create, capture, authorize, and manage PayPal orders via Orders API v2 |
| **Payments** | View captured payments, track refunds, sync from PayPal |
| **Refunds** | Issue full and partial refunds on captured payments |
| **Webhooks** | Receive and process PayPal event notifications with signature validation |
| **Subscriptions** | Catalog products, billing plans, and recurring subscriptions |
| **Payouts** | Send mass payments to email, phone, or PayPal ID |
| **Checkout** | Public payment page for customers to pay without admin access |
| **Dashboard** | Stats widget showing payment totals |

## Requirements

- PHP 8.3+
- Laravel 12.x / 13.x
- Filament v5.x
- PayPal REST API credentials (Client ID + Secret)
